に 偉 ~ る 通り で ある が 、 印 度 の 文化 、 そ の 國民 性 に 就 て は 頃 る 誤 つ た 観察 が 流布 さて わる 。
品 度 は 或 種 の 英 周 人 が 宜 偉 す る や うな 末 開 た 岡 だ ち う か 、 そ れ は 現代 的 文明 に 何 の か はり も
な き 熱 謙 の 亡 財 に 過ぎ な だ らち うか 。

アメ リカ を 僧 せ た より 多く 、 支 那 に 導 ぐ 世界 の 大 國 で あお る 。

上 央 洲 が 末 だ 女 化 が 開け ず 、 無 知 職 味 の 時 代 に 、 印 度 は 既に 最高 の 文明 に 到 差 し て わた 。 そ の
文化 の 思 澤 を 被 ら ちな か つた 周 は 無い で あら う 。 現 代 の 印度 も 決し て それ 以下 に は な つて 己 な ぁ

和 印度 は 同上 思 に 於 ける 六 つ の 大 宗 数 の うち 、 そ の 二 つ を 生ん だ 。 又 、 世 界 の 七 大 鉱 事 詩 の うち そ
